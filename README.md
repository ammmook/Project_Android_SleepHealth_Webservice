## ⚙️ เกี่ยวกับระบบ (About The Service)

Repository นี้คือส่วนของ **Web Service (Backend)** สำหรับแอปพลิเคชัน **Sleep Health** ทำหน้าที่เป็นศูนย์กลางในการประมวลผล Logic ที่ซับซ้อน เชื่อมต่อฐานข้อมูล และส่งข้อมูลกลับไปยังแอปมือถือ

> *"ถ้าแอปมือถือคือหน้าตา Web Service นี้ก็คือสมอง... ที่ทำงานหนักแม้ในยามที่คุณหลับ"* 🧠💤

### 🚀 หน้าที่หลัก (Core Functions)

* 📡 **RESTful API:** ให้บริการ Endpoints สำหรับรับ-ส่งข้อมูล (JSON) กับ Android App
* 🗄️ **Database Management:** จัดการข้อมูล User และประวัติการนอนทั้งหมดลงใน MySQL
* 🏆 **Ranking System:** ประมวลผลและจัดอันดับ Leaderboard ประจำวัน
